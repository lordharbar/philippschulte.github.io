Me, Myself, and I

I am a full-stack developer with 2 years of experience in developing scalable, maintainable, and secure Web applications. I love to build single-page applications, breaking and fixing code, creating REST APIs and writing unit tests right from the start.

Outside of full-time work I love to teach the fundamentals of web development. I used to be an instructor at DevMountain, a full-stack Javascript bootcamp located in Salt Lake City, UT. Currently I am an instructor at Codify Academy, a front-end bootcamp in San Francisco, CA.

I particularly enjoy learning about new JavaScript standards/libraries/frameworks and currently experiment with Elixir and Phoenix. In my free time I experiment with microcontrollers to build my own robots and drones. My specialties are: JavaScript, Elixir, Node, React, Redux, Express, Phoenix, MongoDB, Postgres.

Project Process

Develop

Full-Stack development of your project. This stage varies greatly between each individual project but often includes lots of code, data sources, late nights and plenty of coffee.

Test

Module testing occurs throughout the development process. Once development is complete I recommend conducting a thorough usability test to iron out any creases.

Launch

The big day for both of us! I set-up your project as agreed and am available to oversee your project launch, provide support and enjoy a nice cold beer to celebrate!

My Tools

Node — Server

I use Node with all supported ES6 and beyound features to build full-stack web and command-line applications. MongoDB is my preferred database, able to power anything from my growing personal project to an enterprise infrastructure.

Vue — Browser

I have worked with many JS frameworks and libraries. I love to build components to break down the complexity of my applications. Vue.js is definitely my most favorite one because it combines the best of React and Angular 2.0.

React Native — Mobile

I have build hybrid apps using Ionic and Angular. React Native uses the same component design as React and the same fundamental UI components as regular iOS and Android apps. This allows us to build apps with a native look and performance.
